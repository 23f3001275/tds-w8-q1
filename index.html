Analytics: Earnings Growth Function (Python)


def earnings_growth(current_earnings, previous_earnings):
    """
    Compute period-over-period earnings growth.
    """
    if previous_earnings == 0:
        raise ValueError("Previous earnings cannot be zero.")
    return (current_earnings - previous_earnings) / previous_earnings

# Example: Q3 vs Q2
q3 = 12.4  # in ₹ billion
q2 = 11.0  # in ₹ billion
growth = earnings_growth(q3, q2)
print(f"Q3 earnings growth: {growth:.2%}")

Financial Metrics – ROE & NIM

Return on Equity (ROE):

\[ \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \]

Net Interest Margin (NIM):

\[ \text{NIM} = \frac{\text{Interest Income} - \text{Interest Expense}} {\text{Average Earning Assets}} \]

For this quarter, we target: \[ \text{ROE} \approx 15\% \quad \text{and} \quad \text{NIM} \approx 3.8\% \]

Walk through each formula slowly. - Emphasize that ROE measures profitability relative to equity. - NIM shows how efficiently we generate interest income. Connect these to the strategic goals outlined earlier.
